Work In Progress (WIP)
Produzione
›
Produzione
Module › Production › Transaction › Work In Process › WIP
Formula 1 Process I/P Qty
PI = (OB × SI) / SB
Formula 2 Produced Qty
PQ = (PI × SO) / SI
SB = Standard Batch Size  ·  SI = Standard I/P Qty  ·  SO = Standard O/P Qty
OB = Order Batch Size  ·  PI = Process I/P Qty  ·  PQ = Produced Qty
  • Compilare i campi in ordine; selezionare Process Code in ordine numerico
  • Impostare Process I/P Qty → Formula 1
  • Settare Produced Qty → Formula 2
  • Status → Completed
  • Aggiungere macchinari usati + compilare End Time
  • Aggiungere Cost Component + compilare Actual Hour
  • Selezionare Operator Name e Production Chemist Name → Save
  • Ripetere per ogni processo FG; per processi successivi lasciare Process I/P Qty a zero

Finished Product Transfer Note
Produzione
›
Produzione
Module › Production › Transaction › Finished Product Transfer Note
  • Selezionare Batch No corretto e inserire data
  • Sezione Details Of Full Final Packs: calcolare Quantity Per Pack con la formula sotto, inserire il valore in uno dei quattro campi → Add
  • Sezione Details Of Full Partial Packs: inserire 0 in un campo → Add
  • Salvare
Formula
Quantità = Quantità Prodotta (WIP Confezionamento) ÷ Unit
Esempio
Unit = "1 conf. 200 ml" → totale = 1 × 200 = 200
Quantità Prodotta nel WIP (Confezionamento) = 400
Quantità = 400 ÷ 200 = 2

  • La stampa mostra l'allocazione in rack per ciascuna materia prima emessa
Impostare la resa in eccesso del prodotto finito
›
Config. › General Configuration
  • Inserire la password supervisore quando richiesta
  • Selezionare Production e impostare la percentuale di Finished Goods Excess Yield
  • Attualmente impostata al 900%, per consentire di produrre più unità di quanto richiesto dall'ordine
Produrre più dell'ordine cliente: unire ordine cliente e ordine proprio
›
Esempio: il cliente ordina 2 flaconi di shampoo, ma si decide di produrne 5 in totale: 2 per il cliente e 3 da tenere in farmacia per la vendita. Si creano quindi due ordini separati (2 + 3) e si uniscono in un unico batch da 5.
Step 1: Customer Purchase Order
  • Creare il Customer Purchase Order per la quantità richiesta dal cliente (es. 2 flaconi), nel campo Order Type selezionare Customer Order
  • Creare un secondo Customer Purchase Order per la quantità aggiuntiva da produrre per la farmacia (es. 3 flaconi), nel campo Order Type selezionare Own Order
Step 3: Batch Creation
  • In Batch Creation, aggiungere entrambi gli ordini nello stesso batch
  • Il batch coprirà la quantità totale (es. 5 flaconi)
